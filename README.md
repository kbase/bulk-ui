
# KBase Bulk UI (under dev)

This is a prototype bulk import UI for KBase.
The API consumed is in progress at https://github.com/kbase/kb-ftp-api.


## Requirements

node


## Local Installation

```
git clone https://github.com/kbase/bulk-ui.git
cd bulk-ui
npm install
```

## Start Dev Server

